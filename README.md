# HoloSpot v5.1 🌟

Uma rede social inovadora focada em **destacar pessoas** através de "holofotes".

## 🎯 Conceito

O HoloSpot é diferente de outras redes sociais - aqui o foco é **reconhecer e destacar outras pessoas**, promovendo positividade e valorização mútua na comunidade.

## ✨ Funcionalidades Principais

### 🏠 Feed Principal
- Posts de holofotes em tempo real
- Sistema de reações e comentários
- Filtros por tipo de destaque
- Infinite scroll

### ⭐ Sistema de Destaque
- Criação de posts destacando pessoas
- Tipos de destaque (profissional, pessoal, conquista, etc.)
- Sistema de menções com @username
- Upload de imagens

### 👤 Perfis Completos
- Páginas de perfil com URLs compartilháveis
- Dashboard de estatísticas e insights
- Sistema de níveis e gamificação
- Conquistas e emblemas

### 🔔 Notificações em Tempo Real
- Notificações de menções, seguidores, reações
- Sistema de badges
- Modal de detalhes

## 🏗️ Tecnologias

- **Frontend:** HTML5 + CSS3 + JavaScript (Vanilla)
- **Backend:** Supabase (PostgreSQL + Auth + Real-time)
- **Hosting:** GitHub Pages / Netlify
- **Analytics:** Google Analytics 4
